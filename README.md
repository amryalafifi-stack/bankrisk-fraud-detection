## Local Installation

```bash
# Install dependencies
pip install -r requirements.txt

# Run the application
streamlit run app.py
```

## Usage

1. Upload your CSV file with transaction data
2. Map columns to the required fields (amount, customer ID, category, etc.)
3. Click "Initiate Analysis" to detect fraudulent transactions
4. Review flagged transactions in the audit queue

## Technology Stack

- **Frontend:** Streamlit
- **Data Processing:** Pandas, NumPy
- **Visualization:** Plotly
- **AI Engine:** Custom fraud detection algorithms
